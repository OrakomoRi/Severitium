About
What is
Severitium?

A comprehensive visual overhaul that transforms the Tanki Online interface — without touching gameplay mechanics.

What exactly is Severitium?
Severitium is a visual customization project that transforms the appearance of Tanki Online — a browser-based multiplayer tank game. Think of it as a "skin" that changes how the game looks without affecting gameplay mechanics.
What does it actually do?
The original game interface can feel outdated. Severitium replaces its visual elements with modern, animated designs featuring vibrant colors, smooth transitions, and improved UI components. A complete visual makeover.
How does it work technically?
The project uses CSS and JavaScript to override the game's default appearance. When installed via Tampermonkey, it automatically applies these visual changes whenever you open Tanki Online in your browser.
Who can use it and how difficult is installation?
Any Tanki Online player. No programming knowledge required — just install Tampermonkey and click the script installation link. Two steps total.
Why is this valuable from a dev perspective?
Demonstrates advanced front-end skills: CSS animations, JavaScript architecture, cross-browser compatibility, UX design, and the ability to reverse-engineer and enhance existing complex interfaces.
